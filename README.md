# agenda-de-contatos
Projeto em java para criação de uma agenda de contatos.

Tecnologias utilizadas:
- Servidor Tomcat;
- Java EE (J2EE) - Servlet e JSP;
- IDE Eclipse;
- JDBC (Java Database Connectivity);
- MySQL CRUD;
- MVC (Model View Controller);
- JavaBeans;
- DAO; e
- iText.
